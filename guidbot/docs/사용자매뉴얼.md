## 1. AI 가이드봇이란?

**AI 가이드봇**은 좋은문화병원의 각종 규정집·지침서를 AI가 읽고 학습하여, 직원 여러분의 질문에 즉시 답변하는 AI 도우미입니다.

### 주요 기능

| 기능 | 설명 |
|------|------|
| **규정집 질의응답** | "연차 기준이 어떻게 돼요?", "NEDIS 코드 G1이 뭔가요?" 등 자연어로 질문 |
| **병동 현황 대시보드** | 실시간 병상 현황, 진료과별 재원 환자, 수술 통계 확인 |
| **원무 현황 대시보드** | 수납 현황, 보험별 매출, 미수 통계, 외래 환자 통계 확인 |

### AI가 학습한 내용

- 병원 원무·행정 규정집
- 간호업무지침
- 의약품 관리 지침
- NEDIS(국가응급진료정보망) 코드표 및 가이드
- 각 부서별 업무 지침서
- Oracle DB 매뉴얼 (IT 담당자용)

> **참고**: AI는 학습된 문서를 기반으로 답변합니다.  
> 규정이 개정되어 지식베이스가 업데이트되지 않은 경우, 최신 내용이 반영되지 않을 수 있습니다.  
> 중요한 사항은 원문 규정집을 반드시 확인하세요.

## 3. AI 규정 챗봇 사용법

### 3-1. 화면 구성

```
┌─────────────────────────────────────────────────────┐
│  [사이드바]              │   [채팅 화면]              │
│                          │                            │
│  ☰ 좋은문화병원 가이드봇  │  ┌─────────────────────┐  │
│                          │  │  AI 답변 영역        │  │
│  🔍 검색 모드 선택        │  │  (스트리밍 출력)     │  │
│  ○ ⚡ 빠른 검색          │  └─────────────────────┘  │
│  ● ⚖️ 표준 검색 (기본)   │                            │
│  ○ 🧠 심층 검색          │  참고 문서: [원무규정.pdf] │
│                          │                            │
│  📄 회람문서 바로가기     │  ┌─────────────────────┐  │
│  ─────────────────       │  │  질문 입력창         │  │
│  서비스 링크              │  │  [전송 버튼]         │  │
│  병동 대시보드            │  └─────────────────────┘  │
│  원무 대시보드            │                            │
│  관리자 대시보드          │                            │
└─────────────────────────────────────────────────────┘
```

### 3-2. 기본 사용법

**1단계**: http://192.1.1.231:8502 접속

**2단계**: 검색 모드 선택 (처음에는 기본값 ⚖️ 표준 검색 권장)

**3단계**: 하단 채팅창에 질문 입력

**4단계**: 전송(→) 버튼 클릭 또는 Enter 키

**5단계**: AI가 답변을 실시간으로 생성합니다 (2~5초 소요)

---

### 3-3. 사이드바 기능

#### 검색 모드 선택

| 모드 | 아이콘 | 응답 시간 | 용도 |
|------|--------|----------|------|
| 빠른 검색 | ⚡ | 0.5~1.5초 | 자주 묻는 간단한 질문 |
| 표준 검색 | ⚖️ | 2.5~4초 | **기본 추천** — 일반적인 규정 문의 |
| 심층 검색 | 🧠 | 4~7초 | 복잡한 예외 조항, 판정 기준 문의 |

#### 대화 초기화

- 사이드바 하단 **"대화 초기화"** 버튼 클릭 → 대화 기록 삭제, 새 대화 시작

#### 회람문서 바로가기

사이드바의 **"회람문서 바로가기"** 버튼 클릭 → 병원 공용 드라이브의 최근 회람문서로 이동합니다.

---

### 3-4. 답변 화면 설명

AI가 답변을 생성하면 다음 정보가 표시됩니다:

```
┌──────────────────────────────────────────────────────┐
│ [⚖️ Hybrid 표준검색]                                  │
│                                                       │
│ 연차 유급휴가는 1년간 80% 이상 출근한 직원에게         │
│ 15일이 부여됩니다. 3년 이상 근속 시 2년마다 1일씩      │
│ 추가되어 최대 25일까지 부여됩니다.                    │
│                                                       │
│ 📄 참고 문서                                          │
│  · 직원복무규정.pdf (3조 2항)                         │
│  · 인사관리지침.pdf (연차 산정 기준)                   │
└──────────────────────────────────────────────────────┘
```

- **[배지]**: 사용된 검색 모드 표시 (⚡ 빠른검색 / Hybrid 표준검색 / Multi-Query 심층검색)
- **답변 본문**: AI가 규정집을 분석하여 생성한 답변
- **📄 참고 문서**: 답변의 근거가 된 규정집 파일명과 해당 조항

> **참고 문서가 표시되지 않는 경우**: 질문한 내용이 현재 학습된 규정집에 없는 경우입니다.  
> 다른 표현으로 다시 질문하거나, 직접 원문 규정집을 확인해 주세요.

## 6. 검색 모드 선택 가이드

AI 챗봇의 검색 모드를 상황에 맞게 선택하면 더 정확한 답변을 얻을 수 있습니다.

### ⚡ 빠른 검색 (fast)

**사용 시기**:
- "연차 며칠이에요?" 처럼 단순하고 빠른 답변이 필요할 때
- 자주 묻는 기본 규정을 다시 확인할 때

**예시 질문**:
```
연차 기준 알려줘
식대 지원 금액이 얼마야?
야간 수당 계산 방법은?
```

---

### ⚖️ 표준 검색 (standard) — **기본 권장**

**사용 시기**:
- 규정 내용을 정확하게 확인하고 싶을 때
- 대부분의 일반적인 규정 문의

**예시 질문**:
```
병가 신청 절차와 필요 서류가 뭔가요?
초과근무 신청은 어떻게 하나요?
NEDIS 코드 W5가 어떤 경우에 사용되나요?
퇴직금 계산 기준을 알려주세요.
```

---

### 🧠 심층 검색 (deep)

**사용 시기**:
- 예외 조항이나 특수 사례 확인이 필요할 때
- 두 개 이상의 규정이 관련된 복잡한 질문
- 표준 검색 결과가 충분하지 않을 때

**예시 질문**:
```
계약직 직원의 연차 예외 규정은 어떻게 되나요?
교통사고 환자에 대한 NEDIS 판정 기준과 예외 사항은?
산전·산후 휴가 중 연차 산정에서 제외되는 기간이 있나요?
의료급여 환자의 선택 진료 적용 예외 조항을 알려주세요.
```

## 7. 잘 쓰는 질문 방법

AI가 더 정확하게 답변할 수 있도록 질문 방법을 참고해 주세요.

### 7-1. 구체적으로 질문하기

| 막연한 질문 ❌ | 구체적인 질문 ✅ |
|--------------|----------------|
| "휴가 알려줘" | "육아휴직 신청 기간과 필요 서류가 뭔가요?" |
| "수당 기준" | "야간 근무(22시~06시) 수당 계산 방법을 알려주세요" |
| "NEDIS 오류" | "NEDIS G1 코드 입력 시 판정 오류가 나는 기준은?" |

### 7-2. 상황을 설명하면 더 좋은 답변

```
✅ 좋은 예시:
"계약직으로 입사한 지 6개월 된 직원인데, 
 병가를 신청하려고 합니다. 가능한가요?"

✅ 좋은 예시:
"외래 환자가 당일 예약 없이 응급으로 왔을 때 
 NEDIS 신고는 어떻게 처리하나요?"
```

### 7-3. 핵심 키워드 포함

규정집에서 사용하는 공식 용어를 사용하면 더 정확한 검색이 됩니다.

```
연차유급휴가, 초과근무, 식대지원, 야간근무수당, 
NEDIS 코드, 의료급여, 건강보험, 선택진료, 
산전후휴가, 육아휴직, 병가, 경조사 휴가
```

### 7-4. 이전 대화 활용

챗봇은 이전 대화를 기억합니다. 연속 질문이 가능합니다:

```
[질문 1] 연차유급휴가 기준 알려주세요
[답변]   ...15일 기본 부여...

[질문 2] 그럼 입사 첫 해에는 어떻게 계산해요?
[답변]   1년 미만 재직자의 경우...

[질문 3] 계약직도 동일하게 적용되나요?
```

> 대화 기록은 최대 **15턴**까지 유지됩니다.  
> 주제가 바뀔 때는 "대화 초기화" 버튼을 누르면 더 정확한 답변을 받을 수 있습니다.

## 8. 자주 묻는 질문 (FAQ)

### Q. AI가 "관련 문서를 찾을 수 없습니다"라고 답변합니다.

**A.** 질문한 내용이 현재 학습된 규정집에 없는 경우입니다.
- 다른 표현이나 키워드로 다시 질문해 보세요.
- **심층 검색 모드(🧠)**로 변경 후 다시 시도해 보세요.
- 원문 규정집을 직접 확인하거나 담당 부서에 문의해 주세요.

---

### Q. AI 답변이 정확하지 않거나 최신 규정과 다릅니다.

**A.** AI는 학습된 규정집 기준으로 답변합니다.
- 규정 개정 후 지식베이스가 아직 업데이트되지 않았을 수 있습니다.
- 중요한 인사·행정 결정은 반드시 **원문 규정집 또는 담당 부서를 통해 확인**해 주세요.
- IT 담당자에게 지식베이스 업데이트를 요청하세요.

---

### Q. 답변이 너무 오래 걸립니다.

**A.** 다음을 확인해 주세요:
- **⚡ 빠른 검색** 모드로 변경해 보세요 (0.5~1.5초)
- 병원 내부 Wi-Fi 또는 유선 연결 상태 확인
- 동시에 많은 직원이 사용 중일 때 일시적으로 느릴 수 있습니다. 잠시 후 다시 시도해 주세요.

---

### Q. 개인정보나 환자 정보를 입력해도 되나요?

**A.** **절대 입력하지 마세요.**
- 챗봇에는 환자 이름, 주민번호, 진단명 등 개인정보를 입력하지 않습니다.
- 챗봇은 규정·지침 정보 조회 용도로만 사용합니다.

---

### Q. 병동 대시보드 데이터가 보이지 않습니다.

**A.** Oracle DB 연결 문제일 수 있습니다.
- 화면 상단의 연결 상태가 🔴로 표시되는지 확인
- IT 담당자에게 즉시 문의해 주세요.

---

### Q. 모바일(스마트폰)에서 접속할 수 있나요?

**A.** 네, 병원 내부 Wi-Fi에 연결된 스마트폰/태블릿에서 접속 가능합니다.
- Chrome 또는 Safari 브라우저 사용 권장
- 화면이 좁은 경우 사이드바는 상단 ☰ 버튼으로 열 수 있습니다.

---

### Q. 챗봇에게 대화 기록을 지워달라고 요청할 수 있나요?

**A.** 사이드바 하단의 **"대화 초기화"** 버튼을 직접 클릭하세요.
- 현재 세션의 대화 기록이 삭제됩니다.
- 브라우저를 새로 고침하거나 창을 닫아도 초기화됩니다.

## 9. 이용 주의사항

### 사용 목적

- AI 가이드봇은 **병원 내부 규정 및 지침 안내**를 위한 도구입니다.
- 의학적 진단, 처방, 치료에 관한 질문에는 답변하지 않습니다.

### 개인정보 보호

- 챗봇 입력창에 **환자 개인정보**(이름, 주민번호, 진단명 등) 절대 입력 금지
- **직원 개인정보**(급여 세부 내역, 징계 기록 등) 입력 금지
- AI는 병원 외부 서버(Google Gemini)로 질문 내용을 전송하여 처리합니다.  
  → **민감한 정보는 일반적인 표현으로 변환하여 질문하세요.**

> ❌ 잘못된 예: "홍길동 간호사의 2025년 연차 잔여일수가 몇 일이에요?"  
> ✅ 올바른 예: "입사 3년차 간호사의 연차 잔여 계산 방법이 어떻게 되나요?"

### AI 답변의 한계

- AI는 학습된 문서 기반으로 답변하므로 **오류가 있을 수 있습니다.**
- 인사·징계·계약 등 중요한 의사결정은 반드시 **담당 부서에 직접 확인**하세요.
- AI 답변은 **참고 자료**로만 활용하고, 최종 판단은 사람이 해야 합니다.

### 접속 에티켓

- 동시에 많은 직원이 AI 챗봇을 사용하면 느려질 수 있습니다.
- 응급하지 않은 질문은 **피크 시간대(오전 9~10시, 점심 시간)를 피해** 이용하면 더 빠릅니다.
